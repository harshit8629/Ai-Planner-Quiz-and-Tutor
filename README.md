# Ai-Planner-Quiz-and-Tutor
AI Planner, Quiz and Tutor is an intelligent web-based study assistant designed to help students plan their studies, test their knowledge, and understand concepts using AI-powered conversations.

The platform integrates AI-driven study planning, automated quiz generation, and an AI tutor chatbot to create a personalized learning experience.

This project is built using Python Flask for the backend and HTML, CSS, and JavaScript for the frontend, with AI integration for smart interaction and content generation.

Features
Smart Study Planner

Create a structured study schedule based on selected subjects and topics.

Helps students manage their time efficiently.

Automatically organizes topics into a learning timeline.

AI Quiz Generator

Generates quizzes based on the study topics.

Includes multiple-choice questions to test understanding.

Helps students evaluate their knowledge after studying.

AI Tutor Chat

Interactive chatbot to help explain concepts.

Students can ask questions about their topics.

Provides simplified explanations to improve understanding.

Personalized Learning Experience

Combines planning, learning, and testing in one platform.

Makes studying more organized and interactive.

How It Works

The user enters the subject and study topics.

The system generates a study planner for structured learning.

After studying, the system creates a quiz based on the topics.

If the user does not understand a concept, they can ask the AI Tutor Chat for explanations.

This creates a complete Learn → Practice → Understand cycle.
